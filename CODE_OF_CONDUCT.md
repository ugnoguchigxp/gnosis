# Code of Conduct

このプロジェクトは、参加者全員が安心して協力できる場であることを重視します。

## 私たちが目指す振る舞い

- 敬意を持って対話する
- 具体的で建設的なフィードバックを行う
- 技術的な意見の違いと個人攻撃を切り分ける
- ドキュメント、Issue、PR、レビューで文脈を丁寧に共有する
- 初学者や新規参加者にも配慮する

## 許容しない行為

- 侮辱、差別、ハラスメント、脅迫
- 個人攻撃、人格否定、挑発的な言動
- 意図的な荒らし、スパム、議論の妨害
- 秘匿情報や個人情報の無断公開
- 技術的議論を装った攻撃的・排他的な振る舞い

## 適用範囲

この行動規範は、Issue、Pull Request、コードレビュー、チャット、ドキュメント、その他プロジェクトに関連するあらゆるやり取りに適用されます。

## 運用方針

メンテナーは、プロジェクトの健全性を守るために次の対応を行うことがあります。

- 注意
- 編集依頼
- コメントの非表示または削除
- 一時的または恒久的な参加制限

## 報告方法

問題がある場合は、公開スレッドでのエスカレーションではなく、可能であればメンテナーへ非公開で連絡してください。脆弱性報告については `SECURITY.md` を参照してください。

## 補足

本ファイルは Contributor Covenant の考え方を参考に、Gnosis 向けに簡潔化したものです。
