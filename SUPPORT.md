# Support

Gnosis に関する問い合わせの使い分けです。

## どこに何を出すか

- **バグ報告**: Issue
- **機能要望**: Issue
- **使い方の質問**: Issue または README / docs の改善提案
- **セキュリティ問題**: `SECURITY.md` に従って非公開で報告
- **設計相談や大きな変更案**: まず Issue で背景を共有してから PR

## Issue を作る前に

- README を確認する
- `docs/` 配下に同じ話題の設計メモがないか確認する
- 既存 Issue / PR と重複していないか確認する
- 再現手順や期待値を整理する

## あると助かる情報

- 実行環境（OS、Bun、Python、Docker）
- 実行したコマンド
- ログやエラーメッセージ
- `.env` で変更した主要設定
- 再現手順

## ドキュメント改善も歓迎

Gnosis は README や `docs/` の整備を重視しています。動作自体の不具合でなくても、「分かりにくかった」「導線が弱い」という報告は歓迎します。
